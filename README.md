The project implements a transport management system that models drivers, passengers, and vehicles using abstraction, inheritance, 
and interfaces. The system allows dynamic user input to create transport entities and calculate fares based on distance. 
Exception handling ensures the program handles invalid inputs safely without crashing.On the Transport System some of the exceptions that have been implemented include
the Input mismatch exception which is a commmon exception whereby the user can enter the wrong data type.Other Exceoptions include the Number format Exception.
